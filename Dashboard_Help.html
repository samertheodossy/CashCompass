Introduction

This dashboard reads and updates your Google Sheets (INPUT and SYS tabs). Use Run Planner + Refresh Snapshot to sync latest values into the overview and run the debt planner.

Numbers come from your spreadsheets; this UI does not store its own copy of balances.

Overview

Snapshot cards show totals for cash, investments, real estate, debt, and net worth. Some cards include change vs prior month (or week, where noted)—compares live sheet totals to a prior period from INPUT history or OUT - History as documented in your project notes.

Click a card to jump to the related workspace tab when that shortcut exists.

Assets

House Values — Edits INPUT - House Values for the selected month. SYS - House Assets Current Value is refreshed from the latest values in the current year block when you run the planner (other columns on SYS, such as Type and Loan Amount Left, are not overwritten by that sync).

Bank Accounts and Investments — Update INPUT sheets; SYS tabs reflect synced balances.

Cash Flow

Quick Payment — Adds to INPUT - Cash Flow for the selected year. The info box can show last month’s payment for the same payee: it reads the prior calendar month’s cell on the correct year’s Cash Flow tab (January uses the previous year’s tab for December).

Upcoming — Planned expenses from your sheet; Bills Due is described below.

Bills Due (Cash Flow tab)

Three groups: Overdue, Next 7 Days, and Recurring Bills (No Due Date). Each card shows only the bill name, amount, and due date (where applicable), plus Pay and Skip.

Pay opens Quick Payment with payee, due date, and amount pre-filled. You can change the amount there before saving—there is no separate “suggested” line on the card; the amount shown is the value the dashboard is using (same number Pay starts with).

Where amounts come from — Rows from INPUT - Bills use the bill fields from that sheet. Items tied to INPUT - Debts (minimum payments) use the planner/debt amount; the card does not list category, autopay, or “varies” flags—those stay on the sheet for your reference.

Recurring (no due date) — Bills that are not yet tied to a dated row in INPUT - Bills appear in the third section. Pay opens Quick Payment with today’s date and amount 0 so you can enter the real payment; map or add due dates in INPUT - Bills when you want them in the dated lists.

Skip marks the bill skipped in the dashboard flow (per your backend rules) and refreshes the lists. Feedback from Pay / Skip (for example “Bill skipped… wrote 0 into Cash Flow”) appears in the status line under the Bills Due heading, not next to Run Planner.

Properties

House Expenses — Adds rows to the HOUSES - … tabs (per property).

Property performance — See the next section for how rent, expenses, and the Type column on SYS - House Assets work together.

Property performance (detail)

Rent — The table sums Income rows on INPUT - Cash Flow for the selected year where the Payee starts with Rent plus the same house name as in SYS - House Assets (for example Rent Oakley House - …).

Rent only for rentals — On SYS - House Assets, the Type column controls this: values Rental (and the common typo Renal) use Cash Flow rent. Types such as Home or Vacation Home show $0 rent in the table even if Income rows exist. If Type is blank, rent is still summed (legacy behavior).

Expenses — These come from each property’s HOUSES - … sheet. The table sums Cost + Service Fees for the selected year from the sheet whose location matches that house (matching is case- and spacing-insensitive).

HOUSES sheet — There must be a HOUSES - … tab that lines up with the House name on SYS for those expenses to show. If there is no matching HOUSES sheet—or the tab name doesn’t match the SYS house (for example SD Condo vs San Diego House)—expense totals stay $0 until you add or rename the HOUSES sheet so it aligns.

Net — Rent minus expenses for that row; the portfolio mini-cards sum the same figures across all houses.

Planning

Debts — Update INPUT - Debts; planner output uses balances and minimums.

Retirement and Purchase Sim — Use the latest snapshot and your inputs; details depend on your sheet configuration.

Sheet names (quick reference)

  • INPUT - House Values — Year blocks, monthly valuations per house.
  • SYS - House Assets — One row per property: House, Type, Loan Amount Left, Current Value (and sync targets for value).
  • HOUSES - {House} — Expense detail per property; name should match SYS for expense rollups.
  • INPUT - Cash Flow {year} — Monthly columns; Rent payees for rental income.
  • INPUT - Bills — Dated bills (category, autopay, varies, etc.); drives most Bills Due cards.
  • INPUT - Debts — Loan accounts; minimum payments can appear on Bills Due.
